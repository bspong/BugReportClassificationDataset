GetMethod.java checks the used flag which cannot be set at this time
GetMethod.getResponseBodyAsStream calls HttpMethodBase.checkUsed which asserts the flag used is true . But at this time used cannot be true as used is set to true in HttpMethodBase.processRequest two lines after readResponse is called which in turn calls readResponseBody readResponseBodyAsStream .
