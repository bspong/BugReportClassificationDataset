
Step to generate word-based model

1. Java process [WekaProcessing]

	- Convert Text files in directory to weka instances
	- Apply StringToWordVector
 	- Write to .arff file

2. Manually edit value of attribute 'classify' in .arff from 'string' to '{FEATURE,BUG}'

3. Mining data by using Weka

	- manually remove some noise (click edit button)	

	a. go to Select attributes tab
	- choose attribute evaluator: SymmetricalUncertAttributeEval
	- set selected number to {20, 50, 100}
	- save reduced data

	b. Classify tab
